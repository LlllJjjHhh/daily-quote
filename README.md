# Daily Quote

A beautiful minimalist daily quote web page. Get inspired every day!

🔗 Live Demo: https://<your-username>.github.io/daily-quote/

## Features

- ✨ Minimalist clean design
- 🌓 Auto dark/light mode based on system preference
- 📱 Fully responsive for mobile devices
- 🔄 Random quote on every refresh
- 📋 One-click copy to clipboard
- 💫 Smooth animations

## Usage

1. Fork this repository
2. Enable GitHub Pages in repo settings
3. Done! Your daily quote page is live

Or run it locally:

```bash
git clone https://github.com/<your-username>/daily-quote.git
cd daily-quote
# Just open index.html in your browser
```

## Customization

Edit `quotes.json` to add your favorite quotes:

```json
[
  {
    "text": "Your quote here",
    "author": "Author Name"
  }
]
```
